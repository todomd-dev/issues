# todomd — public bug tracker

This is the public bug tracker for **[todomd](https://todomd.dev)** — the visual task board that dispatches coding agents in isolated git worktrees.

The source code is private. This repo exists only so users can report bugs and request features without hitting a 404.

## Reporting a bug

From inside a project, run:

```
todomd bug "short description of what happened"
```

That opens an issue here with environment context attached. Or just [open an issue manually](https://github.com/todomd-dev/issues/new/choose).

Please include: todomd version (`todomd version`), your runner (`cmux` / `tmux`), OS, and the license email you used to purchase (for order cross-reference) if applicable.

## Feature requests

Use [Discussions](https://github.com/todomd-dev/issues/discussions) for ideas and questions; reserve issues for reproducible bugs.
